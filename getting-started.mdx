## üöÄ **Quick Start**

<Steps>
<Step>

### **Choose Your Path**

Decide where you want to start based on your current experience level:

- **Beginner**: Start with [Core Concepts](/modules/foundations) if you're new to AI/LLM development
- **Intermediate**: Jump to [Prompting & Interaction](/modules/prompting-structured-outputs) if you understand basic AI concepts
- **Advanced**: Explore [AI Architecture](/modules/agents-orchestration) for sophisticated system design

</Step>
<Step>

### **Set Up Your Environment**

Ensure you have the necessary tools and environment ready:

<CodeGroup>

```bash npm install
npm install -g @mintlify/cli
```

```bash yarn add
yarn global add @mintlify/cli
```

</CodeGroup>

</Step>
<Step>

### **Start Learning**

Begin with the recommended modules and work through the content systematically. Each module builds upon the previous ones.

</Step>
<Step>

### **Practice & Apply**

Use the hands-on exercises, code examples, and capstone project to apply what you learn to real-world scenarios.

</Step>
</Steps>
