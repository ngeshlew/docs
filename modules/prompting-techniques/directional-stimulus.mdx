# Directional Stimulus

<Callout type="info">
  **Learning Objective**: Master directional stimulus techniques that guide AI responses toward specific outcomes, behaviors, or thinking patterns through strategic prompt design.
</Callout>

## Overview

Directional Stimulus is a prompting technique that uses specific cues, instructions, or contextual elements to guide AI responses in desired directions. This approach leverages psychological and behavioral principles to influence how AI systems think, respond, and behave.

<CardGroup cols={2}>
  <Card title="Response Guidance" icon="compass">
    Uses strategic cues to direct AI responses toward specific outcomes or behaviors.
  </Card>
  <Card title="Behavioral Control" icon="target">
    Influences AI thinking patterns and response styles through contextual stimuli.
  </Card>
</CardGroup>

## What is Directional Stimulus?

Directional Stimulus is a technique that:

- **Guides Responses**: Uses specific cues to direct AI toward desired outcomes
- **Influences Behavior**: Shapes AI response patterns and thinking styles
- **Controls Context**: Provides contextual stimuli that affect AI decision-making
- **Enhances Precision**: Increases the likelihood of getting specific types of responses

<Callout type="warning">
  **Key Insight**: Directional stimulus works by providing contextual cues that activate specific patterns in the AI's training, leading to more targeted and predictable responses.
</Callout>

```python
class DirectionalStimulus:
    def __init__(self):
        self.stimulus_templates = {
            'role_based': {
                'expert': "You are an expert {domain} with {years} years of experience.",
                'consultant': "Act as a {domain} consultant providing strategic advice.",
                'teacher': "You are a patient teacher explaining {topic} to a beginner.",
                'analyst': "As a data analyst, examine this information critically."
            },
            'emotional': {
                'empathetic': "Respond with empathy and understanding.",
                'enthusiastic': "Show enthusiasm and excitement in your response.",
                'professional': "Maintain a professional and objective tone.",
                'friendly': "Be warm and friendly in your communication."
            },
            'process': {
                'step_by_step': "Think through this step by step.",
                'multiple_perspectives': "Consider multiple perspectives and viewpoints.",
                'evidence_based': "Base your response on evidence and facts.",
                'creative': "Approach this with creativity and innovation."
            },
            'outcome': {
                'concise': "Provide a concise and focused response.",
                'detailed': "Give a comprehensive and detailed analysis.",
                'actionable': "Provide actionable recommendations.",
                'educational': "Make your response educational and informative."
            }
        }
    
    def create_stimulus(self, stimulus_type: str, **kwargs) -> str:
        """Create a directional stimulus based on type and parameters"""
        
        if stimulus_type in self.stimulus_templates:
            template = self.stimulus_templates[stimulus_type]
            
            if isinstance(template, dict):
                # Select appropriate template based on kwargs
                key = kwargs.get('subtype', list(template.keys())[0])
                if key in template:
                    return template[key].format(**kwargs)
                else:
                    return template[list(template.keys())[0]].format(**kwargs)
            else:
                return template.format(**kwargs)
        
        return ""
    
    def build_prompt(self, base_prompt: str, stimuli: list) -> str:
        """Build a complete prompt with directional stimuli"""
        
        # Combine all stimuli
        stimulus_text = "\n".join(stimuli)
        
        # Build the complete prompt
        complete_prompt = f"{stimulus_text}\n\n{base_prompt}"
        
        return complete_prompt
    
    def apply_role_stimulus(self, role: str, domain: str = "", years: int = 10) -> str:
        """Apply role-based directional stimulus"""
        return self.create_stimulus('role_based', subtype=role, domain=domain, years=years)
    
    def apply_emotional_stimulus(self, emotion: str) -> str:
        """Apply emotional directional stimulus"""
        return self.create_stimulus('emotional', subtype=emotion)
    
    def apply_process_stimulus(self, process: str) -> str:
        """Apply process-oriented directional stimulus"""
        return self.create_stimulus('process', subtype=process)
    
    def apply_outcome_stimulus(self, outcome: str) -> str:
        """Apply outcome-oriented directional stimulus"""
        return self.create_stimulus('outcome', subtype=outcome)

# Example usage
stimulus = DirectionalStimulus()

# Create different types of stimuli
role_stimulus = stimulus.apply_role_stimulus('expert', domain='machine learning', years=15)
emotional_stimulus = stimulus.apply_emotional_stimulus('empathetic')
process_stimulus = stimulus.apply_process_stimulus('step_by_step')
outcome_stimulus = stimulus.apply_outcome_stimulus('actionable')

# Build complete prompt
base_prompt = "Explain how to implement a machine learning model for customer segmentation."
complete_prompt = stimulus.build_prompt(base_prompt, [
    role_stimulus,
    emotional_stimulus,
    process_stimulus,
    outcome_stimulus
])

print(complete_prompt)
```

```javascript
class DirectionalStimulus {
    constructor() {
        this.stimulusTemplates = {
            roleBased: {
                expert: "You are an expert {domain} with {years} years of experience.",
                consultant: "Act as a {domain} consultant providing strategic advice.",
                teacher: "You are a patient teacher explaining {topic} to a beginner.",
                analyst: "As a data analyst, examine this information critically."
            },
            emotional: {
                empathetic: "Respond with empathy and understanding.",
                enthusiastic: "Show enthusiasm and excitement in your response.",
                professional: "Maintain a professional and objective tone.",
                friendly: "Be warm and friendly in your communication."
            },
            process: {
                stepByStep: "Think through this step by step.",
                multiplePerspectives: "Consider multiple perspectives and viewpoints.",
                evidenceBased: "Base your response on evidence and facts.",
                creative: "Approach this with creativity and innovation."
            },
            outcome: {
                concise: "Provide a concise and focused response.",
                detailed: "Give a comprehensive and detailed analysis.",
                actionable: "Provide actionable recommendations.",
                educational: "Make your response educational and informative."
            }
        };
    }
    
    createStimulus(stimulusType, kwargs = {}) {
        if (this.stimulusTemplates[stimulusType]) {
            const template = this.stimulusTemplates[stimulusType];
            
            if (typeof template === 'object') {
                // Select appropriate template based on kwargs
                const key = kwargs.subtype || Object.keys(template)[0];
                if (template[key]) {
                    return template[key].replace(/\{(\w+)\}/g, (match, key) => kwargs[key] || '');
                } else {
                    return template[Object.keys(template)[0]].replace(/\{(\w+)\}/g, (match, key) => kwargs[key] || '');
                }
            } else {
                return template.replace(/\{(\w+)\}/g, (match, key) => kwargs[key] || '');
            }
        }
        
        return "";
    }
    
    buildPrompt(basePrompt, stimuli) {
        // Combine all stimuli
        const stimulusText = stimuli.join('\n');
        
        // Build the complete prompt
        const completePrompt = `${stimulusText}\n\n${basePrompt}`;
        
        return completePrompt;
    }
    
    applyRoleStimulus(role, domain = "", years = 10) {
        return this.createStimulus('roleBased', { subtype: role, domain, years });
    }
    
    applyEmotionalStimulus(emotion) {
        return this.createStimulus('emotional', { subtype: emotion });
    }
    
    applyProcessStimulus(process) {
        return this.createStimulus('process', { subtype: process });
    }
    
    applyOutcomeStimulus(outcome) {
        return this.createStimulus('outcome', { subtype: outcome });
    }
}

// Example usage
const stimulus = new DirectionalStimulus();

// Create different types of stimuli
const roleStimulus = stimulus.applyRoleStimulus('expert', 'machine learning', 15);
const emotionalStimulus = stimulus.applyEmotionalStimulus('empathetic');
const processStimulus = stimulus.applyProcessStimulus('stepByStep');
const outcomeStimulus = stimulus.applyOutcomeStimulus('actionable');

// Build complete prompt
const basePrompt = "Explain how to implement a machine learning model for customer segmentation.";
const completePrompt = stimulus.buildPrompt(basePrompt, [
    roleStimulus,
    emotionalStimulus,
    processStimulus,
    outcomeStimulus
]);
```

### 2. **Advanced Directional Stimulus with LangChain**

<Card title="LangChain Integration">
  <CodeGroup>
    <CodeGroupItem title="LangChain Implementation" active>
```python
from langchain.prompts import PromptTemplate
from langchain.chains import LLMChain
from langchain.llms import OpenAI
from typing import Dict, List, Any
import json

class AdvancedDirectionalStimulus:
    def __init__(self, api_key: str):
        self.llm = OpenAI(api_key=api_key, temperature=0.7)
        self.stimulus_configs = {}
        self.response_patterns = {}
    
    def configure_stimulus(self, name: str, config: Dict[str, Any]):
        """Configure a specific directional stimulus"""
        self.stimulus_configs[name] = config
    
    def create_stimulus_prompt(self, stimulus_name: str, **kwargs) -> str:
        """Create a prompt with specific directional stimulus"""
        
        if stimulus_name not in self.stimulus_configs:
            return ""
        
        config = self.stimulus_configs[stimulus_name]
        
        # Build stimulus based on configuration
        stimulus_parts = []
        
        # Add role-based stimulus
        if 'role' in config:
            role_template = config['role']
            stimulus_parts.append(role_template.format(**kwargs))
        
        # Add emotional stimulus
        if 'emotion' in config:
            emotion_template = config['emotion']
            stimulus_parts.append(emotion_template.format(**kwargs))
        
        # Add process stimulus
        if 'process' in config:
            process_template = config['process']
            stimulus_parts.append(process_template.format(**kwargs))
        
        # Add outcome stimulus
        if 'outcome' in config:
            outcome_template = config['outcome']
            stimulus_parts.append(outcome_template.format(**kwargs))
        
        return "\n".join(stimulus_parts)
    
    def create_chain_with_stimulus(self, stimulus_name: str, base_template: str, **kwargs):
        """Create a LangChain with directional stimulus"""
        
        # Create stimulus
        stimulus = self.create_stimulus_prompt(stimulus_name, **kwargs)
        
        # Combine stimulus with base template
        full_template = f"{stimulus}\n\n{base_template}"
        
        # Create prompt template
        prompt = PromptTemplate(
            template=full_template,
            input_variables=kwargs.get('input_variables', [])
        )
        
        # Create chain
        chain = LLMChain(llm=self.llm, prompt=prompt)
        
        return chain
    
    def apply_stimulus_and_generate(self, stimulus_name: str, base_prompt: str, 
                                  inputs: Dict[str, Any], **kwargs) -> str:
        """Apply stimulus and generate response"""
        
        # Create chain with stimulus
        chain = self.create_chain_with_stimulus(
            stimulus_name, 
            base_prompt, 
            **kwargs
        )
        
        # Generate response
        response = chain.run(inputs)
        
        return response
    
    def compare_stimuli(self, base_prompt: str, inputs: Dict[str, Any], 
                       stimulus_configs: List[str]) -> Dict[str, str]:
        """Compare different stimuli on the same prompt"""
        
        results = {}
        
        for stimulus_name in stimulus_configs:
            if stimulus_name in self.stimulus_configs:
                response = self.apply_stimulus_and_generate(
                    stimulus_name, base_prompt, inputs
                )
                results[stimulus_name] = response
        
        return results

# Example usage
advanced_stimulus = AdvancedDirectionalStimulus("your-api-key")

# Configure different stimulus types
advanced_stimulus.configure_stimulus("expert_consultant", {
    "role": "You are an expert {domain} consultant with {years} years of experience.",
    "emotion": "Maintain a professional and confident tone.",
    "process": "Think through this systematically and provide evidence-based recommendations.",
    "outcome": "Provide actionable, strategic advice that can be implemented immediately."
})

advanced_stimulus.configure_stimulus("empathetic_teacher", {
    "role": "You are a patient and understanding teacher explaining {topic}.",
    "emotion": "Show empathy and encouragement throughout your explanation.",
    "process": "Break down complex concepts into simple, digestible parts.",
    "outcome": "Ensure the student feels confident and capable after your explanation."
})

# Test different stimuli
base_prompt = "Explain the concept of {concept} to someone who is new to the field."

inputs = {"concept": "machine learning"}

# Compare responses with different stimuli
results = advanced_stimulus.compare_stimuli(
    base_prompt, 
    inputs, 
    ["expert_consultant", "empathetic_teacher"]
)

for stimulus_name, response in results.items():
    print(f"\n{stimulus_name.upper()} RESPONSE:")
    print(response)
```

```python
from crewai import Agent, Task, Crew
from langchain.tools import Tool
import json

class DirectionalStimulusCrew:
    def __init__(self):
        self.stimulus_agents = {}
        self.response_agents = {}
    
    def create_stimulus_agent(self, name: str, stimulus_config: dict) -> Agent:
        """Create an agent with specific directional stimulus"""
        
        # Build backstory with stimulus
        backstory_parts = []
        
        if 'role' in stimulus_config:
            backstory_parts.append(stimulus_config['role'])
        
        if 'personality' in stimulus_config:
            backstory_parts.append(stimulus_config['personality'])
        
        if 'expertise' in stimulus_config:
            backstory_parts.append(stimulus_config['expertise'])
        
        backstory = " ".join(backstory_parts)
        
        # Create agent
        agent = Agent(
            role=stimulus_config.get('role_title', 'Specialist'),
            goal=stimulus_config.get('goal', 'Provide helpful assistance'),
            backstory=backstory,
            verbose=True,
            allow_delegation=False
        )
        
        self.stimulus_agents[name] = agent
        return agent
    
    def create_response_agent(self, name: str, response_style: dict) -> Agent:
        """Create an agent with specific response style"""
        
        backstory = f"""
        You are a {response_style.get('role', 'assistant')} who specializes in 
        {response_style.get('style', 'helpful')} responses. Your communication style is 
        {response_style.get('tone', 'professional')} and you focus on 
        {response_style.get('focus', 'providing value')}.
        """
        
        agent = Agent(
            role=response_style.get('role', 'Assistant'),
            goal=response_style.get('goal', 'Provide excellent assistance'),
            backstory=backstory,
            verbose=True,
            allow_delegation=False
        )
        
        self.response_agents[name] = agent
        return agent
    
    def create_stimulus_task(self, agent_name: str, task_description: str, 
                           stimulus_config: dict) -> Task:
        """Create a task with directional stimulus"""
        
        # Build task description with stimulus
        task_parts = [task_description]
        
        if 'process' in stimulus_config:
            task_parts.append(f"\nProcess: {stimulus_config['process']}")
        
        if 'outcome' in stimulus_config:
            task_parts.append(f"\nExpected Outcome: {stimulus_config['outcome']}")
        
        if 'constraints' in stimulus_config:
            task_parts.append(f"\nConstraints: {stimulus_config['constraints']}")
        
        full_task_description = "\n".join(task_parts)
        
        return Task(
            description=full_task_description,
            agent=self.stimulus_agents[agent_name]
        )
    
    def run_stimulus_comparison(self, task_description: str, 
                              stimulus_configs: dict) -> dict:
        """Compare different stimuli on the same task"""
        
        results = {}
        
        for stimulus_name, config in stimulus_configs.items():
            # Create agent with stimulus
            agent = self.create_stimulus_agent(stimulus_name, config)
            
            # Create task
            task = self.create_stimulus_task(stimulus_name, task_description, config)
            
            # Create crew
            crew = Crew(
                agents=[agent],
                tasks=[task],
                verbose=True
            )
            
            # Execute
            result = crew.kickoff()
            results[stimulus_name] = result
        
        return results

# Example usage
stimulus_crew = DirectionalStimulusCrew()

# Define different stimulus configurations
stimulus_configs = {
    "expert_analyst": {
        "role": "You are a senior data analyst with 10+ years of experience in business intelligence.",
        "role_title": "Senior Data Analyst",
        "personality": "You are analytical, detail-oriented, and focused on data-driven insights.",
        "expertise": "You excel at interpreting complex data and providing actionable business recommendations.",
        "goal": "Provide data-driven insights and strategic recommendations.",
        "process": "Analyze the data systematically, identify key patterns, and provide evidence-based conclusions.",
        "outcome": "Deliver clear, actionable insights that drive business decisions.",
        "constraints": "Focus on practical, implementable recommendations."
    },
    
    "creative_consultant": {
        "role": "You are a creative consultant who specializes in innovative problem-solving.",
        "role_title": "Creative Consultant",
        "personality": "You are imaginative, open-minded, and always looking for novel approaches.",
        "expertise": "You excel at thinking outside the box and finding creative solutions to complex problems.",
        "goal": "Generate innovative ideas and creative solutions.",
        "process": "Explore multiple perspectives, consider unconventional approaches, and think creatively.",
        "outcome": "Provide unique, innovative solutions that stand out from conventional approaches.",
        "constraints": "Ensure solutions are both creative and practical."
    }
}

# Compare different stimuli
task_description = "Analyze the current market trends in AI and provide recommendations for a startup."

results = stimulus_crew.run_stimulus_comparison(task_description, stimulus_configs)

for stimulus_name, result in results.items():
    print(f"\n{stimulus_name.upper()} RESULT:")
    print(result)
```
